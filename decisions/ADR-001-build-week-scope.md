# ADR-001 Build Week Scope

- Status: Accepted
- Date: 2026-07-19

## Context

Generic personalized novel product is too broad for challenge window and already has close competitors.

## Decision

Build one open-source local LitRPG demo in original reincarnated Demon King world. User locks one of six existing viewpoints. No hosted or commercial features.

## Reason

Narrow scenario makes persistent world motion, viewpoint isolation, and GPT-5.6 agent use visible and testable.

## Consequences

- Next.js, TypeScript, local SQLite, MIT, user API key.
- No auth, billing, social layer, mobile, or other genres.
