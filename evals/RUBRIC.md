# Narrative Review Rubric

Score 0 to 2. Cite text evidence. Any zero in continuity, POV safety, LitRPG mechanics, or arc progress blocks release.

| Dimension          | 0                        | 1               | 2                               |
| ------------------ | ------------------------ | --------------- | ------------------------------- |
| Choice fulfillment | Ignored                  | Weak effect     | Clear causal effect             |
| Character autonomy | Background actors inert  | Generic motion  | Distinct goal-driven motion     |
| POV safety         | Hidden fact leaked       | Ambiguous       | Knowledge boundary clean        |
| LitRPG mechanics   | Contradiction            | Thin or unclear | Consistent and meaningful       |
| Continuity         | Canon break              | Minor strain    | State and history preserved     |
| Arc progress       | Stalled or rushed        | Partial         | Milestone pressure fits clock   |
| Prose              | Repetitive or incoherent | Readable        | Strong voice and scene movement |

Human release owner makes final call. Reviewer agents provide evidence only.
