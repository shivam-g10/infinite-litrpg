## Three-Minute Demo

Use the timed [recording script](DEMO_SCRIPT.md). Required spoken evidence includes both Codex collaboration and GPT-5.6 use.

1. Show six characters and lock Rowan viewpoint.
2. Choose action.
3. Show up to three Luna background intents.
4. Show World Director resolve one valid delta.
5. Replay the audited Luna chapter as a stream containing only Rowan-safe knowledge.
6. Open God Mode trace, cost, latency, and act clock.
7. Show meaningful-decision continuation, chapter-100 stop, and chapter-350 terminal tests.

## Submission Evidence

- Public GitHub repository.
- Public YouTube video under three minutes.
- Setup instructions.
- Architecture diagram.
- Test and eval output.
- One complete trace.
- Sanitized current non-live gate output.
- Screenshots of reader and inspector.
- Codex feedback session ID covering core build.

Official rules require a public YouTube video with audio. Repository may be public with a license, or private and shared with both judge accounts listed in the [submission packet](SUBMISSION.md).

## Ownership

- User owns narrative signoff, feedback session ID, publication approval, video approval, and final submission authority.
