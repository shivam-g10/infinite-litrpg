# Build Week

- Track: Apps for Your Life
- Deadline: 2026-07-22 05:30 IST
- Demo thesis: Choose one life. World keeps moving without you.

## Three-Minute Demo

1. Show six characters and lock Rowan viewpoint.
2. Choose action.
3. Show up to three Luna background intents.
4. Show World Director resolve one valid delta.
5. Stream Luna chapter containing only Rowan-safe knowledge.
6. Open God Mode trace, cost, latency, and act clock.
7. Show chapter 350 terminal test.

## Submission Evidence

- Public GitHub repository.
- Public YouTube video under three minutes.
- Setup instructions.
- Architecture diagram.
- Test and eval output.
- One complete trace.
- Screenshots of reader and inspector.
- Codex feedback session ID covering core build.

## Ownership

- Agent owns build, tests, evals, docs, screenshots, demo script, and evidence packet.
- User owns narrative signoff, feedback session ID, publication approval, video approval, and final submission authority.
